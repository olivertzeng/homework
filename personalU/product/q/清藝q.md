# 未來學習計畫與生涯規劃（Q）

## 一、近期規劃（錄取至入學前）

1. 作品整理與記錄：將過去的摺紙作品、木製展示架與 LED 互動裝置進行完整的影像記錄與製作過程文字整理，建立個人作品集的初始版本。
2. 互動技術預習：自學 Arduino 或 Raspberry Pi 的基礎操作，為入學後的互動裝置課程做準備，銜接我過去在紅外線感測模組上的實作經驗。

## 二、中期規劃（大學期間）

1. 補足藝術論述能力：大一大二階段廣泛接觸數位影像、聲音藝術、互動裝置等不同媒材的創作方法。我目前最大的缺口不在技術，而在「為什麼要做這個作品」的論述能力。我會做出能動的東西，但還不會說明它為什麼值得被看。這是我在科藝組最需要學習的核心能力。
2. 展覽實務：參與校內外展覽策劃，學習如何為自己的作品建構完整的藝術脈絡，從「能做出來」進化為「能說明為什麼這樣做」。
3. 開源精神的藝術應用：延續我在 GitHub 上的開源貢獻習慣，將創作過程中開發的互動工具或程式模組開源分享，探索科技藝術領域中的協作與共創可能性。

## 三、遠期規劃（畢業後）

畢業後期望成為科技藝術領域的創作者與技術開發者。我計畫持續創作結合程式運算、實體裝置與觀眾互動的作品，並參與國內外數位藝術展覽與駐村計畫。長期目標是建立一個能讓工程師與藝術家共同協作的開源互動藝術平台。
