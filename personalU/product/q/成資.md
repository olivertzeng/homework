# 未來學習計畫與生涯規劃（Q）

### 一、近期規劃（錄取至入學前）

1. 補強數理基礎：高中階段的物理表現不如預期，這讓我確認自己必須在基礎科學上投入更多心力。入學前我將專注預習微積分與線性代數，為大二的演算法與機率統計打底。
2. 演算法實作轉型：利用 ZeroJudge 等平台持續解題，但目標從「把功能寫出來」轉向「分析時間與空間複雜度後再動手」，為銜接系上的演算法課程做準備。

### 二、中期規劃（大學期間）

1. 系統與網路底層課程：大一大二除了扎實必修外，我會特別投入「計算機組織」「作業系統」與「電腦網路概論」。我在設定 Docker bridge network、配置 Tailscale 跨裝置 VPN 時反覆碰壁的問題，根源都在於缺乏對封包從應用層到網路層完整路徑的系統性理解，這些課程正好針對這個盲區。
2. 爭取加入分散式系統相關實驗室：以系上 115 級專題為例，莊坤達教授指導的 Kubernetes DSCP 封包優先級控制、Apache Ozone RocksDB 壓縮最佳化等主題，都需要對容器化環境、網路協定與系統底層有實務理解。我過去三年自架伺服器與維護十餘個 Docker 服務的經驗，雖然不具備學術層級的嚴謹度，但能提供進入實驗室後快速上手的實務基礎。我的目標是將過去單打獨鬥的系統除錯經驗，轉化為實驗室層級的研究方法論。
3. 持續開源貢獻：運用過去 145 個 PR 的跨國協作經驗，嘗試參與更大規模的開源基礎設施專案（如 Docker 周邊工具、容器網路相關專案），將課堂學到的理論直接應用於真實的開源社群中。

### 三、遠期規劃（畢業後）

畢業後期望成為專注於基礎設施或分散式系統的軟體工程師。我計畫投入雲端資源調度與系統底層開發，並持續以開源貢獻者的身分，開發能服務技術社群的系統工具。
