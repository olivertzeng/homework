# 未來學習計畫與生涯規劃（Q）

## 一、近期規劃（錄取至入學前）

1. 網路基礎預習：入學前透過 Jim Kurose 的 Computer Networking 官方 YouTube 講解系列，把目前只靠實務摸索理解的 TCP/IP、DNS、NAT 等概念補上理論框架。我目前對這些協定的理解是「知道設定檔怎麼改」但不知道「為什麼要這樣改」，這個落差需要在入學前開始補。
2. 持續開源貢獻與英語輸出：保持在 GitHub 國際社群的活躍度，維持技術英語的讀寫語感，為全英語授課環境做銜接。

## 二、中期規劃（大學期間）

1. 深耕網路與系統底層：大一大二階段全力投入計算機網路概論、資料結構與演算法等必修，從根本理解我在設定 Tailscale、配置反向代理時只能靠直覺處理的底層機制。大三起鎖定計算機組織與微算機概論，補足硬體層面的知識。在七選四的專業選修中，我優先選擇作業系統與軟體工程，將過去零散的系統管理經驗轉化為正規的軟體工程實踐。
2. 善用國際研修資源：積極爭取參加 IPAS 等電資學院國際研討會，延續我在 GitHub 上跨國協作的經驗。四年級的國際研修必修，是我將大學所學帶入國際場域的關鍵機會。

## 三、遠期規劃（畢業後）

畢業後期望成為專注於網路基礎設施與系統安全的工程師。我計畫投入雲端架構、網路安全或 DevOps 等領域，將自架伺服器的實戰經驗與大學四年的正規訓練結合，持續以開源精神開發基礎設施工具。
